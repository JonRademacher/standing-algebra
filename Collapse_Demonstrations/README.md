# Structural Collapse Demonstrations

This folder contains executable demonstrations of a real class of
system failure: compositional collapse from individually legitimate actions.

For each validation system there is a general collapse followed by what such collapses look like when integrated with 
Standing Algebra. Specific case examples are subdivided by folders for ease of use. I started with a notoriously
difficult collapse structure to diagnose and resolve in that of Goodhart Drift. 

These demonstrations require no assumptions about:
- coordination
- truthfulness
- intent
- system size

They operate purely on structural invariants.
